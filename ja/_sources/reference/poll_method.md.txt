## Pollとは？

PMEのPoll Methodは、**今のBlenderの状態でそのメニューやボタンを使ってよいか**
を判定するためのPythonコードです。

- `True` を返す
  - 表示や実行の候補になります
- `False` を返す
  - スキップされます

たとえば、メッシュオブジェクトがアクティブなときだけ使いたい場合は、次のように書けます。

```python
ao = C.active_object; return ao and ao.type == 'MESH'
```

## 先に覚えておくとよいこと

- Pollは「何を実行するか」ではなく、「いつ使えるか」を決めます
- Blenderの `Context` を読むことが中心になります
- `True` / `False` の判定が分かれば、実用的なPollはかなり書けます
- 同じホットキーを条件で使い分けたいときにも役立ちます
