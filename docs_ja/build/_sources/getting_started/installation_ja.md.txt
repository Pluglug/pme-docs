# PME のインストール

Pie Menu Editor (PME) をBlenderにインストールする方法を説明します。

## 前提条件

- Blender 3.0以降
- 管理者権限（システム全体にインストールする場合）

## インストール方法

### 方法1: アドオンとしてインストール（推奨）

1. **PMEアドオンのダウンロード**
   - [GitHub Releases](https://github.com/Pluglug/pie-menu-editor-fork/releases)から最新版をダウンロード
   - `.zip`ファイルをダウンロードしてください

2. **Blenderでのインストール**
   - Blenderを起動
   - `Edit` → `Preferences` → `Add-ons`
   - `Install...`ボタンをクリック
   - ダウンロードした`.zip`ファイルを選択
   - `Install Add-on`をクリック

3. **アドオンの有効化**
   - インストール後、アドオンリストで「Pie Menu Editor」を検索
   - チェックボックスをオンにして有効化

### 方法2: 手動インストール

1. **ファイルの配置**
   ```
   Blenderインストールディレクトリ/
   └── 3.x/
       └── scripts/
           └── addons/
               └── pie_menu_editor/
   ```

2. **権限の設定**
   - ファイルに適切な読み取り権限を設定

## インストール後の確認

1. **アドオンの確認**
   - `Edit` → `Preferences` → `Add-ons`
   - 「Pie Menu Editor」が表示されていることを確認

2. **機能の確認**
   - 3Dビューポートで`F3`キーを押す
   - 「Pie Menu Editor」が検索結果に表示されることを確認

## トラブルシューティング

### よくある問題

**Q: アドオンが表示されない**
A: Blenderを再起動してください。それでも解決しない場合は、アドオンファイルが正しく配置されているか確認してください。

**Q: エラーが発生する**
A: Blenderのバージョンが3.0以降であることを確認してください。

**Q: 機能が動作しない**
A: アドオンが有効化されているか確認してください。
