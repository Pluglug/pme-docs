# Welcome to PME Documentation

Pie Menu Editor (PME) is a Blender add-on originally created by roaoao in 2016. It lets you build pie menus, regular menus, pop-up dialogs, macros, and custom panels without writing code, so you can customize Blender's interface freely.

```{admonition} Note
:class: note

Some pages may still be incomplete or contain information from older versions.

If anything is unclear, please use the [Blender Artists Forum](http://blenderartists.org/forum/showthread.php?392910).
```

```{admonition} Do I need programming knowledge?
:class: hint
No. PME can be used without programming.
If you use Python, you can customize it even further.
```

## Where to Get It

Pie Menu Editor Fork is available on Gumroad. It supports Blender 5.0 and later.

```{button-link} https://pluglug.gumroad.com/l/pie-menu-editor-fork
:color: primary
:shadow:
Buy Pie Menu Editor Fork
```
